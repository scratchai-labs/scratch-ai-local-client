# Scratch AI 教练本地客户端

本上下文描述学生在个人电脑上使用 Scratch 创作，并由第三方 AI 服务提供即时编程辅导的产品领域。

## Language

**单机直连版**:
不依赖项目自建服务器，由学生在本机配置自己的 AI API Key，客户端按需直接请求 AI 服务商的产品形态。
_Avoid_: 完全离线版、服务器版、云端教学版

**自学辅导**:
学生独立创作或解决编程问题时，AI 根据当前 Scratch 作品提供启发式提示的学习过程。
_Avoid_: 课堂教学、教师授课、远程教学

**AI 服务商**:
接收客户端按需发送的作品摘要并生成辅导提示的第三方模型服务；请求不经过本项目自建服务器。
_Avoid_: 教学服务器、平台后端

**作品摘要**:
从当前 Scratch 作品中提取、用于生成辅导提示的结构化上下文，不代表完整教学记录。
_Avoid_: 学生档案、课堂记录、云端作品
