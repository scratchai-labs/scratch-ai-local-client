# Scratch AI 教练本地客户端

本上下文描述学生在个人电脑上使用 Scratch 创作，并由第三方 AI 服务提供即时编程辅导的产品领域。

## Language

**单机直连版**:
不依赖项目自建服务器，由学生在本机配置自己的 AI API Key，客户端按需直接请求 AI 服务商的产品形态。
_Avoid_: 完全离线版、服务器版、云端教学版

**自学辅导**:
学生独立创作或解决编程问题时，AI 根据当前 Scratch 作品提供启发式提示的学习过程。
_Avoid_: 课堂教学、教师授课、远程教学

**AI 服务商**:
接收客户端按需发送的作品摘要并生成辅导提示的第三方模型服务；请求不经过本项目自建服务器。
_Avoid_: 教学服务器、平台后端

**作品摘要**:
从当前 Scratch 作品中提取、用于生成辅导提示的结构化上下文；包含当前角色完整脚本及其他角色的少量结构摘要，但不包含媒体资源或原始项目文件。
_Avoid_: 完整项目、学生档案、课堂记录、云端作品

**主动求助模式**:
仅在学生主动请求帮助时，客户端才向 AI 服务商发送当前作品摘要并生成新提示。
_Avoid_: 手动模式、被动模式

**自动辅导模式**:
客户端检测到当前作品发生有效变化后，直接向 AI 服务商请求并刷新辅导提示。
_Avoid_: 自动提醒模式、后台教学

**辅导会话**:
学生在客户端当前运行期间持续编辑同一个 Scratch 作品的辅导过程；切换作品或退出客户端即结束。
_Avoid_: 账号会话、课堂会话、长期学习记录

**辅导上下文**:
辅导会话内用于保持提示连续性的短期信息，至少包含上一轮提示及其后的作品变化；只保存在内存中。
_Avoid_: 聊天历史、学生档案、持久化记忆

**有效作品变化**:
会改变当前角色程序结构或内容、并可能影响辅导判断的 Scratch 作品变化；选择积木、移动视图或窗口变化不属于有效作品变化。
_Avoid_: 任意界面变化、操作事件

**关键积木提示**:
面向低年级学生，按建议使用顺序展示当前下一步所需的约 3 个 Scratch 积木，每个积木附一句简短中文原因，但不提供额外示例或完整作品脚本。
_Avoid_: 无序候选积木、抽象方向提示、完整答案

**作品驱动辅导**:
学生自由创作，客户端仅根据已经完成的 Scratch 作品内容判断并给出下一步关键积木，不要求学生输入目标或问题。
_Avoid_: 聊天辅导、问答助手、目标表单
