# Scratch AI 教练 · 使用说明（小白版）

> 当前版本：**v0.3.0（测试版）**<br>
> 适用电脑：Windows / Mac

---

> **📢 重要提示**  
> **当前为测试版，软件持续开发更新中。**  
> **建议你自己申请 DeepSeek Key 并填写到软件里，效果会更好。**  
> **使用中有任何问题，欢迎随时联系沟通。**

---

## 1. 这个软件是干什么的？

`Scratch AI 教练` 是一个放在电脑上的小助手。

它会：

1. 帮你打开本机的 Scratch
2. 看你现在在 Scratch 里做了什么
3. 用彩色积木的样子，把“当前角色程序”显示出来
4. 给你“下一步可以怎么做”的小提示，并推荐 1～3 个积木

**一句话：**  
你还是自己在 Scratch 里搭积木；教练窗口只负责“看着、提醒、推荐”，**不会替你改作品**。

---

## 2. 使用前请先准备

- 电脑已经装好了 **Scratch 桌面版**（Scratch Desktop / Scratch 3）
- 下载并打开了 **Scratch AI 教练**
- **建议自己申请一个 DeepSeek Key**  
  填进去后，AI 提示效果会更好  
  **没有 Key 也能用**，只是提示会更基础一些

### 最重要的一条规则

**请一定从“教练软件”里打开 Scratch，不要自己先双击打开 Scratch。**

如果 Scratch 已经自己打开了：

1. 先把 Scratch 完全退出
2. 再回到教练软件
3. 点 **「打开已选 Scratch」**

## 4. 怎么安装

### Windows

**安装版：**

1. 双击 `ScratchDesktopCompanion-setup.exe`
2. 按提示点下一步
3. 安装完成后打开软件
4. 看电脑右下角托盘区，是否出现软件图标

**便携版：**

1. 把 `ScratchDesktopCompanion-portable.exe` 放到一个固定文件夹
2. 双击运行

### Mac

**DMG：**

1. 双击 `ScratchDesktopCompanion-mac.dmg`
2. 把软件拖到“应用程序”
3. 打开软件

**ZIP：**

1. 解压 `ScratchDesktopCompanion-mac.zip`
2. 双击软件打开

> 如果 Mac 提示“无法打开 / 来自未知开发者”：  
> 到 **系统设置 → 隐私与安全性** 里点允许，再打开一次。

## 5. 第一次使用：跟着做就行

请按这个顺序：

```text
打开 Scratch AI 教练
  → 选择 Scratch 软件（如果它没自动找到）
  → 点「打开已选 Scratch」
  → 等状态变成“已连接”
  → （可选）填写本课目标
  → （可选）去设置里填写 DeepSeek Key
  → 开始在 Scratch 里创作，看提示
```

### 第 1 步：打开教练软件

打开后，你会看到主窗口。

### 第 2 步：选择 Scratch

- 如果软件已经自动找到 Scratch，会显示“已选 Scratch”路径
- 如果没有，点 **「选择 Scratch 软件」**

Windows 通常选：

- `Scratch.exe`
- 或 `Scratch 3.exe`

Mac 通常选：

- `Scratch.app`
- 或 `Scratch Desktop.app`
- 或 `Scratch 3.app`

### 第 3 步：点「打开已选 Scratch」

1. 点 **「打开已选 Scratch」**
2. 等大约 3～10 秒
3. 状态变成类似 **「已连接」**

这时说明教练已经连上 Scratch 了。

### 第 4 步（可选）：填写本课目标

如果老师这节课有明确任务，可以在主界面的 **「本课目标」** 里写一句话。

例如：

- `1+100 用重复执行求和，并说出结果`
- `鸡兔同笼：已知头数和脚数，求鸡和兔各有多少`

不填写也能用，软件会自动判断；但做数学题、公式题时，写清楚目标通常会让提示更稳。

### 第 5 步：开始看提示

- 你在 Scratch 里搭积木
- 教练窗口会显示：
  - 本课目标（如果你填了）
  - 当前角色
  - 当前角色程序
  - 下一步提示
  - 推荐积木

## 6. 主界面上这些按钮是干什么的？

| 按钮 | 作用 |
| --- | --- |
| 选择 Scratch 软件 | 告诉教练：你电脑上的 Scratch 在哪里 |
| 打开已选 Scratch | 让教练帮你打开 Scratch 并连接 |
| 重新连接 | 连不上时点一下，再试一次 |
| 生成下一步提示 | 让教练重新给你一条建议 |
| DeepSeek 设置 | 填写 AI 密钥、选择模型、选择自动/手动提示 |

主界面里的 **「本课目标」** 不是按钮，是一个可选输入框：

- 不填：软件会根据 Scratch 作品自动判断你在做什么
- 填了：AI 会优先围绕这节课的目标给提示
- 数学题建议填清楚，例如 `1+100 用重复执行求和，并说出结果`

你还会看到：

- **状态**：有没有连上
- **本课目标**：这节课想完成什么，可填可不填
- **当前角色**：你现在在编哪个角色
- **已选 Scratch**：现在用的是哪个 Scratch
- **下一步提示**：AI 或本地给出的小建议
- **推荐积木**：建议你下一步可以试的积木

## 7. 怎么添加 DeepSeek Key？（强烈建议）

**建议自己申请并填写 Key，效果会更好。**

- 有 Key：提示通常更聪明、更贴合作品
- 没有 Key：也能用，但只会给比较基础的本地提示

### 7.1 先去拿到 Key

1. 打开 DeepSeek 平台：`https://platform.deepseek.com`
2. 注册 / 登录
3. 找到 **API Keys**（密钥）页面
4. 新建一个 Key
5. **马上复制保存好**（关掉后可能再也看不到完整内容）
6. 确认账号有余额

> 提醒：  
> Key 相当于密码，不要发给学生，不要截图发群。

### 7.2 在软件里填写 Key

1. 打开 Scratch AI 教练
2. 点右上角 **「DeepSeek 设置」**
3. 在 **DeepSeek API Key** 输入框里粘贴 Key
4. 先点 **「测试 Key」**
5. 测试通过后，点 **「保存 Key」**
6. 关掉设置窗口
7. 回到主界面，点 **「生成下一步提示」** 试一下

### 7.3 这几个按钮分别做什么

| 按钮 | 作用 |
| --- | --- |
| 测试 Key | 检查这个 Key 能不能用 |
| 保存 Key | 把 Key 记在这台电脑上 |
| 清除 Key | 删除已保存的 Key |

### 7.4 模型怎么选？

| 选项 | 适合谁 |
| --- | --- |
| `deepseek-v4-flash` | 日常首选，更快 |
| `deepseek-v4-pro` | 更强，但可能更慢、更费 |

选好后点 **「保存模型」**。

### 7.5 自动提示，还是手动提示？

| 模式 | 意思 | 建议 |
| --- | --- | --- |
| 自动刷新 | 你改积木后，软件自己更新提示 | 自己探索时可用 |
| 手动点击 | 只有你点“生成下一步提示”才更新 | **课堂更推荐** |

> 保存 Key 后，软件默认会切到“手动点击”，这样更省钱，提示也不会一直乱跳。

### 7.6 怎么知道 Key 已经生效？

- 主界面不再提示“还没有保存 Key”
- 提示来源显示 DeepSeek
- 点“生成下一步提示”后，不再总是“本地基础提示”

如果还不行：

1. 重新打开设置
2. 再测试一次 Key
3. 再保存一次
4. 再点一次“生成下一步提示”

### 7.7 没有 Key 能不能用？

可以。  
软件会给你比较基础的本地提示，保证你还能继续做。  

但如果你希望提示更有用，**还是建议自己申请并填写 Key**。

## 8. 学生怎么用？

1. **先开教练软件**，不要先开 Scratch
2. 确认已经选好 Scratch
3. 点 **「打开已选 Scratch」**
4. 如果老师给了任务，在 **「本课目标」** 里写清楚
5. 在 Scratch 里正常创作
6. 看教练窗口的提示和推荐积木
7. 自己回到 Scratch 里找积木、搭积木
8. 需要时再点 **「生成下一步提示」**
9. 关掉主窗口后，软件通常还在托盘里，没有真正退出

请记住：

- 推荐积木是“建议”，不是标准答案
- 软件不会替你搭积木
- 本课目标只是帮助 AI 理解任务，不会限制你继续改作品
- 换角色后，提示会跟着当前角色变化
- 空白作品通常还没有太多可提示的内容

## 9. 老师 / 机房怎么用？

课前：

1. 每台电脑先装好 Scratch
2. 再装好 Scratch AI 教练
3. 先测试一遍：选择 Scratch → 打开 → 已连接
4. 如果要用更好 AI，提前填好 Key 并测试

课中：

- 学生不要自己双击打开 Scratch
- 如果是统一任务，建议让学生在 **「本课目标」** 填同一句，例如 `1+100 用重复执行求和，并说出结果`
- 连不上就先点“重新连接”
- 建议用“手动点击”生成提示，方便控制节奏

## 10. 常见问题

### Q1：一直提示“请先选择 Scratch 软件”

- Scratch 装好了吗？
- 选的是不是真正的 Scratch 软件，而不是坏掉的快捷方式？
- 再点一次 **「选择 Scratch 软件」**

### Q2：一直提示“请从伴随程序打开已选 Scratch”

说明你还没有通过教练软件打开 Scratch。

处理：

1. 不要自己双击 Scratch
2. 如果 Scratch 已经开着，先退出
3. 回到教练软件，点 **「打开已选 Scratch」**

### Q3：点了打开，但连不上

1. 看 Scratch 有没有真的被打开
2. 点 **「重新连接」**
3. 还不行，就退出 Scratch 和教练软件，再重来一遍

### Q4：当前角色程序是空的

- 这个角色里是不是本来就没有积木？
- 状态是不是已经“已连接”？
- 换个角色看看
- 再点一次“重新连接”

### Q5：没有推荐积木

可能是：

- 作品已经比较完整，软件只是在告诉你怎么玩
- 还没点“生成下一步提示”
- 网络或 Key 有问题，暂时只能给很基础的提示

可以再点一次 **「生成下一步提示」**。

### Q6：为什么显示“本地基础提示”

常见原因：

- 还没保存 Key
- Key 无效
- 余额不足
- 网络不好

去设置里点 **「测试 Key」** 看看。

### Q7：Mac 打不开软件

到 **系统设置 → 隐私与安全性** 允许打开。

### Q8：窗口关了，软件还在吗？

通常还在。  
它会缩到托盘 / 菜单栏。  
想彻底退出，从托盘图标菜单里退出。

### Q9：可以先打开 Scratch，再让教练连上去吗？

**现在不行。**  
请一定从教练软件里打开 Scratch。

## 11. 出问题了，怎么反馈？

当前是**测试版**，遇到问题很正常，**随时联系我就行**。

反馈时尽量带上这些：

- 你的电脑是 Windows 还是 Mac
- 用的是哪个安装包
- 窗口上的状态文字
- 一张截图
- 简单说明：你点了什么，然后发生了什么

日志位置（给会看文件的人）：

**Windows：**

```text
C:\Users\你的用户名\AppData\Roaming\@scratch-ai\desktop-companion\desktop-companion.log
```

**Mac：**

```text
~/Library/Application Support/@scratch-ai/desktop-companion/desktop-companion.log
```

也可以在这里提问题：

`https://github.com/scratchai-labs/scratch-ai-local-client/issues`
