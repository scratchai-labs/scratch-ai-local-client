# Scratch AI 教练 v0.3.0

发布日期：2026-07-17

## 本次更新

- 新增可选的“本课目标”输入。数学题、公式题或明确任务可以直接写下目标，AI 提示会更贴近当前任务；不填写时仍会自动判断。
- 大幅改进鸡兔同笼、1 到 n 累加、平方计算等数学题辅导，减少任务反转和无关运动积木建议。
- 推荐积木升级为 DeepSeek Strict 结构化协议。客户端会校验积木连接关系并自行生成 Scratch XML，不接受 AI 直接提供的原始代码或 XML。
- 推荐积木现在可以正确显示变量名、列表名、循环次数、公式、说话内容、秒数、转角等参数，也能展示条件、循环和分支结构。
- 扩展按键事件、角色点击、运算、变量、侦测等常用积木推荐，并加强复杂目标的下一步建议质量。
- 优化主窗口和 DeepSeek 设置页：异步操作会显示“测试中 / 保存中 / 清除中”，键盘焦点、状态播报和错误提示更清晰。
- 提升 Scratch 启动、重新连接和 Bridge 注入稳定性；修复偶发已启动但无法收到作品状态的问题。
- 加强本地安全与诊断：收紧 Bridge 跨源访问、隔离状态快照、明确提示损坏的本地配置文件。

## 质量与兼容性

- Workspace 自动化测试：349 项通过。
- Scratch Blocks 真实 Renderer 合同：5,103 项通过。
- macOS Electron UI smoke、打包和真实 Scratch E2E 通过。
- 正式 Release 继续只提供无预置 Key 的 Windows portable/setup 和 macOS zip/dmg。
