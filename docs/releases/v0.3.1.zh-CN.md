# Scratch AI 教练 v0.3.1

发布日期：2026-07-20

## 本次更新

- 修复多轮 AI 推荐中变量命名漂移的问题。前一轮已经推荐过 `sum`、`i` 等变量时，后续提示会尽量复用同一变量名，不再轻易变成“求和”“计数”等同义新名字。
- 保留学生在 Scratch 里真实创建或改名的变量。如果作品中已经存在“求和”“计数”等变量，教练不会强行改回旧变量名。
- 推荐归一化层增加变量语义连续性保护，覆盖累加器、计数器、上限、分数、生命值、速度、密码、BMI 等常见变量含义。
- 新增变量连续性 95% 覆盖率门禁，防止后续修改破坏多轮推荐变量命名稳定性。

## 质量与兼容性

- Workspace 自动化测试：377 项通过。
- 变量连续性覆盖率门禁：lines 100%、functions 100%、branches 95.49%。
- 正式 Release 继续只提供无预置 Key 的 Windows portable/setup 和 macOS zip/dmg。
